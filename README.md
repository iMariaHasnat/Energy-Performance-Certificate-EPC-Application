# Energy-Performance-Certificate-EPC-Application

OVERVIEW

The Energy Performance Certificate (EPC) Filtering Application is a desktop software solution designed to manage, filter, and analyse EPC property data efficiently.

Built using C# and Windows Forms, the application allows users to search property records, apply filters, view EPC information, and export results to Excel for reporting and operational use. It integrates with a Microsoft Access database for local data storage and retrieval.

This project demonstrates skills in desktop application development, database integration, and data handling workflows.

---

Features
* Property data filtering
* Search functionality
* Microsoft Access database integration
* Export filtered records to Excel
* User-friendly Windows Forms interface

---

Technologies Used
* C#
* .NET Framework
* Windows Forms
* Microsoft Access
* Microsoft Excel
* Visual Studio
